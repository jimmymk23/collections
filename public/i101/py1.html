Python Programming Terms:

  • Function - Something that carries out a command or task. For example, "print" will print whatever it is supposed to.
  • Argument - Something that is being carried out by the function. For example, 6*217 is the argument and will be calculated with the function print. This is also easy to spot because it will always be inside the function's parenthesis.
  • String - Basically anything that placed within quotation marks is recognized as a string.
  • Variable - Can be anything that points or referrs to something else. For example, word1="supercalifragalisticexpialadocious", where I am now able to just type word1 instead of whatever you've it set to.
  • Integer - A number, as long as it doesn't have a decimal... that's called a "float".

Why is it important to keep track of data types?

This is important because once programs start getting larger and more complex, it may become confusing and more difficult to recall what was a string, what was a variable, what was an integer, etc.

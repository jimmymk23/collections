What is the correct syntax for a while loop?

"While Loops" are great for repeating a series of code that will continue as long as a certain criteria is met. For example, while number = 5:, "number = 5" being the criteria that needs to be met for the loop to work. The code that is directly below it and indented will be carried out as long as the number is still 5.

What is a good use of remove in a program?

A good use for "remove" in a program is when you have a list and want to randomly select items from that list. Using "remove" will make sure that the items already selected cannot be selected again.

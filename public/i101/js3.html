    What is the difference between n = 1 + 2 and n = "1" + "2"? If you were to display the contents on n, what would be shown on your page for each?

  • For n = 1 + 2, n will equal the sum of 1 and 2 which is 3. This is because 1 and 2 are integers which can be added together as typical numbers.
  • For n = "1" + "2", n will equal 12 because the quotations indicate that whatever is between them should be treated as a character, not a value. So when combined, n will just be whatever it is you're adding, strung together next to each other.
